Регистрация

Ваши потребности изменились? Если для ваших рабочих потоков потребуется выпуск PhantomPDF Standard for HP, вы можете установить обновление в любое время. Выпуск Standard является расширенной версией выпуска Express и предусматривает следующие возможности для пользователей.
  • Редактирование содержимого PDF-файлов 
  • Управление PDF-документами (то есть объединение, перемещение, удаление, добавление, выравнивание, обрезка и извлечение страниц)
  • Распознавание отсканированных документов
  • Экспорт в другие файловые форматы 
  • Разработка PDF-форм 
  • Защита PDF-файлов
